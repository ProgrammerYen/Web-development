HyperText Markup Language - HTML

HTML stands for HyperText Markup Language. It is the building blocks to a websites features and it is the skeletal structure of a webpage.

Cascading Style Sheets - CSS

CSS stands for Cascading Style Sheets. It is used for decorating a webpage, creating beautiful visuals and precise layout which is aesthetically pleasing.

Javascript

Javascript controls the behaviour of a webpage. This langauge makes a webpage responsive with functionalities.
