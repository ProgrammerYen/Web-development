HTML Tags

Here is a list of HTML elements.

  • <p> - paragraph
  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> - header elements
  • <li> - presents a certain value in a
  • <ul>/<ol>
    "ul" tells the web browser that an unordered list should be defined
    "ol" tells the web browser that an ordered list should be defined.

Here is an example of an ordered list.

  1. Text1
  2. Text2
  3. Text3

You get the jist

HTML stands for HyperText Markup Language

Of course there are far more elements that html uses and can be found here: MDN HTML Elements Reference.
